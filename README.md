# Millenuits - Application - Compte rendu commercial

![Bannière Millenuits](https://ap-bts-sio-louis.github.io/millenuits/assets/banniere_millenuits.png)

> Application PHP dédiée à la gestion des comptes rendus commerciaux de Millenuits, avec un environnement Docker de test proche de la pré-production MN21.

## 📋 Description
Ce dépôt regroupe l'application de suivi des comptes rendus commerciaux de l'entreprise Millenuits.
Il contient le code source PHP dans le dossier `src` ainsi qu'un environnement `docker-compose` permettant de valider les évolutions techniques et fonctionnelles avant déploiement.

L'objectif est de fournir une base de travail fiable pour :
- structurer la saisie des comptes rendus ;
- faciliter l'exploitation des informations terrain ;
- sécuriser les tests sur un environnement représentatif du serveur de pré-production.

## 🛠 Notions et Compétences abordées
Liste des concepts techniques et savoir-faire mis en œuvre :
- **Développement web PHP** : structuration d'une application métier orientée saisie et consultation de données commerciales.
- **Conteneurisation Docker** : exécution de l'application dans un environnement reproductible (Apache + PHP + MariaDB).
- **Déploiement et validation technique** : préparation et tests sur un contexte proche de la pré-production (MN21).

## 📂 Contenu du dépôt
Ce dépôt contient les ressources suivantes :
- 📄 **Code applicatif** : dossier `src` contenant le code de l'application PHP.
- 📊 **Configuration d'environnement** : fichier `docker-compose.yml` pour lancer les services web et base de données.
- 🔧 **Documentation projet** : ce `README.md` pour présenter le contexte, la structure et les procédures de lancement.
